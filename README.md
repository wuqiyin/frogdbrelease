# FrogDB

[中文](./README.zh.md) | [English](./README.md)

FrogDB is a modern desktop SQL workspace for MySQL teams.
<img width="1024" height="1024" alt="logo" src="https://github.com/user-attachments/assets/eb1acca7-6a0c-466d-898e-a0f94d70a145" />

It combines an editor-first workflow, visual data exploration, and safe execution controls so you can move from idea to validated query faster.

## Product Overview

FrogDB is designed for developers, analysts, and DBAs who work with multiple databases every day.
Instead of treating SQL as isolated query windows, FrogDB provides a project-oriented workspace with reusable connections, tabbed execution context, and quick data navigation.
The goal is simple: reduce friction in daily database operations while keeping control and safety.

## Features

- Multi-connection workspace with persistent connection profiles
- Secure remote access through optional SSH tunnel configuration
- Read-only protection mode that intercepts risky modification statements
- Monaco-powered SQL editor for a familiar, productive coding experience
- Database and table exploration with result panels and row-level detail views
- Script and query workflow support with tabbed project context
- Visual UX enhancements such as search palette, filter tools, and quick navigation dialogs

## Core Advantages

- Faster daily SQL workflow through an integrated workspace instead of fragmented tools
- Better operational safety with built-in read-only controls for sensitive environments
- Lower context switching cost with connections, scripts, and results managed in one UI
- Native desktop performance backed by Go + Wails architecture
- Cross-platform delivery model for teams running mixed operating systems

## Typical Use Cases

- Investigating production issues with safe read-only query sessions
- Developing and validating SQL scripts across multiple schemas
- Exploring unfamiliar database structures with quick table/data inspection
- Managing recurring analysis and ad-hoc query tasks in one reusable workspace

FrogDB is positioned as a productivity-focused MySQL desktop client:
powerful enough for advanced SQL workflows, lightweight enough for everyday use, and opinionated toward safe, efficient execution.

Built with Go, React, TypeScript, and Wails.
